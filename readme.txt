WPerrorFixer Plugin is a tool to fix your WordPress.

== Description ==

WPerrorFixer Plugin is an easy to use plugin designed to provide risk free error fixes. 

== Installation ==

WPerrorFixer Plugin can be installed either via the "Plugins > Add New" screen in the WordPress dashboard, or by downloading the WPerrorFixer Plugin via WordPress.org and uploading the ZIP file through the 'Plugins > Add New > Upload' screen in your WordPress dashboard.
 
If your WordPress screens are not accessible you can upload the zip file /wp-content/plugins/ directly into your WordPress directory.

== Frequently Asked Questions ==

How do you use the WPerrorFixer Plugin?

The WPerrorFixer Plugin assumes that your WordPress is not accessible and so works without using WordPress functions.  In the majority of cases the WPerrorFixer Plugin will find and fix your WordPress error. If the WordPress error is found but cannot fixed automatically, then the WPerrorFixer Plugin will provide simple, straightforward and easy to follow instructions for manually fixing the error. 

On the rare occasions that the WordPress error is not one that the WPerrorFixer Plugin can diagnose or safely fix, it will suggest you visit our website WPerrorFixer.com. 

 The online tool available at WPerrorFixer.com can safely diagnose and fix more complex and unusual errors.  It is also able to diagnose and fix across the older (and occasionally problematic) versions of PHP in addition to  current one.  (The current version of the WPerrorFixer Plugin is designed with the latest version of PHP).WPerrorFixer.com has the ability to take backups, enabling it to back out all or any changes made to ensure the process remains risk free.

= What is the WPerrorFixer Plugin for? =

The WPerrorFixer Plugin is designed to fix the following common errors:

Delete this message:	
briefly unavailable for scheduled maintenance. check back in a minute

Mysql Errors. For example:
MySQL server has gone away
[Table 'wp_options' is marked as crashed and should be repaired]
Error 145
Error Establishing A Database Connection
